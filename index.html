CodeAcross Sacramento 2014 is coming!!!

Code for America and Sunlight Foundation have developed ideas of things for us to do. Which do we like? Want to work on one of them? All of them?

The Open Data Inventory

Take stock of where your city stands on publishing some fundamental datasets.
See CfA google doc here.

Data treasure hunt! Go out and try to locate each of the data sets listed in this google document. If and when you find something, add it to the document.


Open Data Policy

Work with your local officials to create an open data policy, or draft an implementation plan or accountability structure for a recently passed policy.
See CfA google doc here.

Open Civic Data Scraper

Create scrapers to pull down important civic information to feed the Open Civic Data API.
