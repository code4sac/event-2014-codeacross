Welcome to CodeAcross Sacramento 2014!

Didn't this used to be called Code Across America?

Yes it did. For the third year in a row, Code for America has organized events to unite technologists and civic enthusiasts and work on projects that improve communities. The big difference about this year... is that Code for America has gone international! No longer just in America, CodeAcross events are now happening simultaneously across the world, driving civic engagement on a global scale.
